# Question 1

### Integer (int)

Integer data type is used to store whole numbers.

Example:
10, 25, 100

### Float (float)

Float data type is used to store decimal numbers.

Example:
10.5, 25.7

### String (str)

String data type is used to store text or characters.

Example:
"Pradhyumn", "Hello"

### Boolean (bool)

Boolean data type stores only two values: True or False.

Example:
True, False

## Question 4

Five commonly used string methods are:

1. upper() - Converts a string into uppercase.
   Example: "hello" → "HELLO"

2. lower() - Converts a string into lowercase.
   Example: "HELLO" → "hello"

3. replace() - Replaces one character or word with another.
   Example: "cat" → "bat"

4. strip() - Removes extra spaces from a string.
   Example: " hello " → "hello"

5. split() - Splits a string into parts.
   Example: "a,b,c" → ['a', 'b', 'c']
